# Persona

![微信 图标](https://open.weixin.qq.com/zh_CN/htmledition/res/assets/res-design-download/icon64_wx_logo.png)

## 项目介绍  

Persona是一个基于微信开发者工具开发的**性格测试类小程序**。
## 功能简述

程序通过参考MBTI（迈尔斯-布里格斯类型指标）测试及其相关研究设置问题，用户通过回答这些问题，得到一份个人性格测试报告。
